Adds a beautiful responsive coming soon page to your site that lets visitors know your site is in development.

== Description ==

Show your visitors a beautiful and elegant coming soon landing page with countdown timer and subscriber form to collect email addresses.

<blockquote>
For more advanced features, themes and to get guaranteed support be sure to check out our premium version of the plugin, <a href="http://rocketplugins.com/wordpress-coming-soon-plugin/">coming soon landing page</a>
</blockquote>

If your site is ready but you want to make some visual changes like site design modifications. You can enable the maintenance mode plugin with ninja coming soon plugin.

Regular visitors on your site would get a site offline for maintenance page or coming soon landing page. Admins can easily make changes to content, design or overall site functionality without worrying about anything.

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
